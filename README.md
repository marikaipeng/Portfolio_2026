# My Portfolio Website

A simple personal portfolio website with a contact form that stores submissions in Supabase.

## 🌐 Live Demo

**Website:** https://portfolio-2026-ghr5.onrender.com/index.html

---

## Overview

This project contains:
- a responsive frontend portfolio website
- a Node.js + Express backend
- a Supabase database for saving contact messages

Both the frontend and backend run from the same local server at http://localhost:3000.

## Features

- Multi-page portfolio website
- Responsive layout
- Contact form with validation
- Contact submissions saved to Supabase

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: Supabase (PostgreSQL)

## Prerequisites

Make sure these are installed:
- Node.js 18+
- npm
- A Supabase project

## Setup

1. Clone or download this project.
2. Create a .env file in the project root with:

```env
PORT=3000
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
```

> Important: `SUPABASE_SERVICE_ROLE_KEY` must be the Supabase service role key from Project Settings -> API, not the anon/public key.

3. Create the database table in Supabase:

```sql
CREATE TABLE IF NOT EXISTS contacts (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    name TEXT NOT NULL,
    email TEXT NOT NULL,
    message TEXT NOT NULL,
    submitted_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

4. Install dependencies:

```bash
npm install
```

## Run the Project

### Option 1: Windows PowerShell

1. Open PowerShell.
2. Go to the project folder:

```powershell
cd "C:\Users\kaipe\OneDrive\Desktop\Portfolio_2026-main"
```

3. Install dependencies:

```powershell
npm install
```

4. Start the server:

```powershell
node server.js
```

5. Open your browser and go to:

```text
http://localhost:3000
```

You can also use:

```powershell
npm start
```

### Option 2: VS Code Terminal

1. Open the project folder in VS Code.
2. Press Ctrl + ` to open the terminal.
3. Run the same commands:

```powershell
npm install
node server.js
```

4. Open the browser and visit:

```text
http://localhost:3000
```

### If the server does not start

- Make sure the .env file exists and contains valid Supabase values.
- Make sure no other app is already using port 3000.
- If needed, stop the old process and try again.

## What Happens When You Run It

- The portfolio homepage is served from the root URL
- The contact form submits to /contact
- Submitted messages can be viewed at /messages

## Project Structure

- index.html — homepage
- SRC/pages/ — portfolio pages
- SRC/script/script.js — frontend interactions
- SRC/style/style.css — styling
- server.js — backend server
- database/schema.sql — database SQL schema
- .env — local environment variables

## Notes

- Keep your .env file private and do not commit it to GitHub.
- Press Ctrl+C in the terminal to stop the server.
